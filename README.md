# Supplementary material for Patrik et al Coexistence of phenotypic plasticity and habitat choice in natural populations

The repository contains 3 files (in addition to this `README.md`) that contains the code to simulate the data (`1_selkie_field_work.r`), the simulated data (`2_selkie_data.csv`) and the code for the analysis(`3_selkie_analysis.r`)

The `2_selkie_data.csv` file contains 500 observations from 50 individuals with 10 observations each. The files has 4 columns:

- id: anonymized selkie identity
- sex: sex of the individual with male coded as 0 and female as 1
- aggr: aggresiveness measurement
- temp: water temperature when aggressiveness was measured
